# SEVAK: Smart Civic Engagement & Village Administration Kendra

SEVAK is a high-fidelity civic governance platform designed to bridge the gap between citizens, administrative authorities, and field officers. It provides a seamless, AI-powered workflow for reporting, categorizing, and resolving urban and rural infrastructure issues.

## 🚀 Key Portals
1. **Citizen Portal**: An intuitive interface for reporting issues with GPS-locked coordinates and photo verification.
2. **Authority Dashboard**: A centralized command center for administrative officers to track, prioritize, and dispatch tasks.
3. **Officer Portal**: A mobile-optimized task management interface for field workers (like Officer Ravi) to receive and resolve assignments.

## ✨ Core Features
- **AI Priority Engine**: Automatically categorizes issues (e.g., Potholes, Sanitation) and assigns a priority score.
- **Deep GPS Lock**: Ensures reporting accuracy with hardcoded demo support for Navsahyadri College.
- **Global Demo Sync**: Zero-latency synchronization between all portals using localized state management, ensuring a perfect demo experience even without a stable backend.
- **Premium Aesthetics**: A sophisticated "Black-and-Cream" design system with organic UI elements and social connectivity textures.

## 🛠 Tech Stack
- **Frontend**: Next.js 14, Tailwind CSS, Framer Motion, Lucide React.
- **Backend**: Node.js, Express, Vercel AI SDK.
- **Intelligence**: Google Gemini 1.5 Flash / Groq Llama 3.3.
- **Storage**: Supabase (Database) + Global Demo Store (LocalStorage Sync).
